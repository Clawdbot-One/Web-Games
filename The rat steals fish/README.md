# 🐭🐟🐱 老鼠偷吃鱼 - 趣味益智游戏

一款基于HTML5 Canvas的趣味益智游戏，控制小老鼠在迷宫中收集鱼，躲避猫咪的追捕！

## 📖 游戏介绍

在这个游戏中，你将扮演一只机智的小老鼠，目标是在限定时间内收集迷宫中的鱼。但是要小心，猫咪会在迷宫中巡逻，如果你被猫咪抓到，就会失去一条生命！

## 🎮 游戏特色

- **6个精心设计的关卡**：从简单到困难，挑战逐步升级
- **4种强力道具**：帮助你对抗猫咪，更轻松地通关
- **完整的得分系统**：多种得分方式，挑战高分
- **排行榜功能**：与其他玩家比拼，争夺榜首
- **响应式设计**：支持PC端和移动端玩耍
- **无需安装**：直接在浏览器中运行

## 🕹️ 操作方式

### 电脑端

- **方向键**或**WASD键**：移动小老鼠
- **ESC键**或**P键**：暂停/继续游戏

### 移动端

- 使用屏幕上的**虚拟方向键**控制移动

## ⭐ 道具系统

| 道具图标 | 道具名称 | 效果说明             |
| ---- | ---- | ---------------- |
| 🧹   | 魔法扫帚 | 让所有猫咪害怕，暂时停止移动5秒 |
| 💣   | 猫咪炸弹 | 暂时移除屏幕上所有猫咪10秒   |
| ⚡    | 加速药水 | 小老鼠的移动速度提升8秒     |
| 🛡️  | 护盾   | 免疫一次与猫咪的碰撞       |

## 🏅 评级系统

- **完美！**：收集100%的鱼 🎉
- **优秀！**：收集80%以上的鱼 ✨
- **通过**：收集60%以上的鱼 ✓
- **失败**：收集少于60%的鱼，或生命耗尽 ✗

## 💯 得分规则

| 项目     | 得分          |
| ------ | ----------- |
| 每条鱼    | 100分        |
| 每个道具   | 50分         |
| 连击加成   | 连击数 × 10分   |
| 剩余生命奖励 | 每条生命 × 200分 |
| 剩余时间奖励 | 每秒 × 5分     |
| 完美通关加成 | +1000分      |
| 优秀通关加成 | +500分       |

**连击说明**：在2秒内连续吃鱼会触发连击，连续吃鱼越多，加成越高！

## 🚀 如何开始

1. 下载游戏文件到你的电脑
2. 使用浏览器打开 `index.html` 文件
3. 点击「开始游戏」按钮
4. 享受游戏吧！

## 🛠️ 技术实现

- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript (ES6+)**：游戏逻辑
- **Canvas API**：游戏画面渲染
- **localStorage**：排行榜数据存储

## 🎨 设计风格

- **配色方案**：温暖的橙色调为主，代表活力和乐趣
- **视觉元素**：使用可爱的emoji作为游戏角色和道具
- **动效设计**：流畅的动画和过渡效果
- **响应式布局**：适配各种屏幕尺寸

## 💡 游戏建议

1. **观察猫咪路线**：先观察猫咪的移动规律，再制定行动计划
2. **善用道具**：在合适的时机使用道具，可以事半功倍
3. **保持连击**：连续吃鱼可以获得额外分数加成
4. **控制节奏**：不要贪心，有时保存生命更重要
5. **利用传送门**：在高级关卡中，传送门是很好的逃生工具

## 🏆 排行榜说明

- 排行榜保存在浏览器本地
- 支持输入3个字符作为名字（大写字母和常用符号）
- 只保存前100名的最高分
- 游戏结束后，如果分数足够高，可以提交到排行榜
