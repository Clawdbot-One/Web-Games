## 1. 游戏简介

**猫吃鱼大冒险**是一款以经典Pac-Man玩法为基础的休闲街机游戏。玩家扮演一只可爱的小猫咪，在一个充满鱼类的迷宫中探险。游戏融合了动作、解谜和策略元素，让玩家在紧张刺激的追逐中体验乐趣。

### 游戏特色

- 🎮 6个风格迥异的关卡
- 🐱 可爱的猫咪角色
- 🐟 3种不同分值的鱼类
- 🐭 智能AI老鼠系统
- ⚡ 6种强力道具
- 🏆 全球积分排行榜
- 🌟 丰富的评价系统

---

## 2. 游戏目标

游戏的核心目标是在规定时间内吃掉尽可能多的鱼：

- **最低目标**：吃到至少60%的鱼（通过关卡）
- **良好目标**：吃到80%以上的鱼（获得优秀评价）
- **终极目标**：吃掉100%的鱼（获得完美评价）

### 注意事项

⚠️ 老鼠会偷走未被及时吃掉的鱼！
⚠️ 如果吃到的鱼数量低于60%，关卡将以失败告终
⚠️ 合理利用道具可以大大提高通关效率

## 4. 游戏角色

### 4.1 猫咪（玩家角色）

```
       🐱
   ╭───────╮
   │  ◉ ◉  │   可爱的橘猫
   │   ◠   │
   ╰───────╯
```

**特性**：
- 玩家控制的主角
- 可以在迷宫中自由移动
- 可以吃鱼和收集道具
- 受到道具效果影响

**状态效果**：
- 🛡️ **护盾激活**：周围有蓝色光环
- ⚡ **加速状态**：显示闪电图标
- 👻 **隐身状态**：身体变透明

### 4.2 老鼠（敌人）

```
       🐭
   ╭───────╮
   │  • •  │
   │   ◡   │
   ╰───────╯
```

**AI行为模式**：

1. **巡逻模式** 🔍
   - 老鼠随机在迷宫中移动
   - 寻找附近未被吃的鱼

2. **追踪模式** 🎯
   - 发现猫咪后主动追击
   - 当猫咪不在隐身状态时触发

3. **偷鱼模式** 🐟
   - 发现附近有未吃的鱼
   - 会快速接近并偷走鱼类
   - 偷鱼后恢复巡逻

4. **逃跑模式** 💨
   - 被冰冻弹击中后冻结
   - 冻结期间无法移动

### 4.3 鱼类

鱼类是游戏的主要目标，分三种类型：

| 类型 | 外观 | 特点 |
|------|------|------|
| 普通鱼 | 灰色 | 数量最多，容易遇到 |
| 金鱼 | 金色 | 数量较少，价值更高 |
| 能量鱼 | 彩虹色 | 稀有，价值最高 |

```

效果：立即增加30秒游戏时间
```

## 8. 评价系统

### 8.1 评价等级

| 评价 | 条件 | 标志 | 奖励 |
|------|------|------|------|
| 失败 | < 60% | ❌ | 无 |
| 通过 | 60-79% | ✅ | +1,000分 |
| 优秀 | 80-99% | 🌟 | +3,000分 |
| 完美 | 100% | 🏆 | +5,000分 |

### 8.2 评价计算

```
完成率 = 吃到的鱼数量 ÷ 鱼的总数量 × 100%

示例：
- 30条鱼，吃了18条：18/30 = 60% → 通过
- 30条鱼，吃了24条：24/30 = 80% → 优秀
- 30条鱼，全部吃完：30/30 = 100% → 完美
```

### 8.3 完美通关

🏆 **完美通关要求**：
- 吃掉100%的鱼
- 没有任何一条鱼被老鼠偷走
- 这需要速度、策略和道具的完美配合！

## 9. 得分计算

### 9.1 得分公式

```
关卡得分 = 基础得分 + 关卡奖励 + 时间奖励
```

### 9.2 详细计算

#### 基础得分

```
基础得分 = 普通鱼(×100) + 金鱼(×300) + 能量鱼(×500)
```

#### 关卡奖励

| 评价 | 奖励 |
|------|------|
| 失败 | 0分 |
| 通过 | +1,000分 |
| 优秀 | +3,000分 |
| 完美 | +5,000分 |

#### 时间奖励

```
时间奖励 = 剩余时间 × 10
```

### 9.3 得分示例

**场景**：第1关，120秒限制

**玩家表现**：
- 吃到普通鱼：20条（2000分）
- 吃到金鱼：6条（1800分）
- 吃到能量鱼：2条（1000分）
- 剩余时间：45秒

**计算**：
- 基础得分：2000 + 1800 + 1000 = 4,800分
- 完成率：28/30 = 93.3% → 优秀评价
- 关卡奖励：+3,000分
- 时间奖励：45 × 10 = 450分
- **本关总分**：4,800 + 3,000 + 450 = **8,250分**

### 9.4 总分计算

游戏结束后，总得分 = 所有关卡得分之和

## 10. 排行榜

### 10.1 排行榜规则

- 📊 记录前100名玩家的最高分
- 👤 玩家名字：3个大写字母或常用符号（!@#$%）
- 📅 记录日期：游戏日期自动记录
- 🔄 实时更新：新记录自动排序

### 10.2 录入排行榜

当游戏结束时，如果你的得分足够高：

1. 系统显示「新纪录！」提示
2. 输入你的名字（3个字符）
3. 点击「录入排行榜」保存

### 10.3 查看排行榜

在主菜单点击「排行榜」即可查看：
- 所有记录的排名
- 玩家名字
- 得分
- 记录日期

## 11. 游戏技巧

### 11.1 新手技巧

#### 🚀 快速上手

1. **优先吃近处的鱼**：节省时间
2. **注意老鼠位置**：避免被偷鱼
3. **收集道具**：关键时刻可以救命
4. **不要贪心**：安全第一

#### 🎯 优先级

```
高优先级：
- 快要被老鼠发现的鱼
- 高分鱼（能量鱼、金鱼）

中优先级：
- 路径上的普通鱼
- 道具

低优先级：
- 远离老鼠区域的鱼
```

### 11.2 进阶技巧

#### ⚡ 速度控制

- 使用闪电靴快速穿梭
- 在鱼群密集区加速收集
- 用速度优势抢先于老鼠

#### 🛡️ 护盾使用

- 进入老鼠密集区前激活
- 保护高价值鱼群
- 配合闪电靴效果更佳

#### 👻 隐身战术

- 穿越危险区域
- 接近高分鱼时使用
- 躲避追击的老鼠

#### ❄️ 冰冻战术

- 关键时刻救命
- 创造安全收集时间
- 配合猫薄荷效果翻倍

#### 🎪 召唤假猫

- 长时间安全行动
- 需要专注吃鱼时使用
- 分散多只老鼠注意力

### 11.3 关卡通关策略

#### 第1-2关：练手

- 熟悉操作和地图
- 练习走位和躲避
- 尝试完美通关

#### 第3-4关：策略

- 开始使用道具
- 规划收集路线
- 注意老鼠AI增强

#### 第5-6关：挑战

- 合理分配道具
- 预判老鼠行为
- 追求完美评价

### 11.4 常见问题

**Q: 鱼被偷走了怎么办？**
A: 尽快吃掉附近的鱼，减少损失

**Q: 道具栏满了怎么办？**
A: 使用道具或错过收集，道具栏最多6个

**Q: 老鼠一直追我怎么办？**
A: 使用隐身、冰冻或召唤假猫

**Q: 时间快到了怎么办？**
A: 使用时间沙漏或加速道具

**Q: 如何解锁所有关卡？**
A: 每个关卡至少吃到60%的鱼
